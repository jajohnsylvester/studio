# **App Name**: Fiscal Flow

## Core Features:

- Expense Entry: Manually input expenses with descriptions and amounts.
- Budget Creation: Set monthly budget limits for different spending categories.
- Categorization Tool: The tool applies AI to categorize each expense into predefined categories (e.g., food, transportation, entertainment) automatically.
- Spending Visualization: Generate charts (bar, pie) to visualize spending across categories.
- Progress Tracking: Show a clear visual representation of budget vs. actual spending for each category.
- Insight Generation: The tool uses AI to offer personalized financial tips based on spending patterns.
- Dashboard Summary: A summary screen showing total income, expenses, and remaining budget.

## Style Guidelines:

- Primary color: Medium blue (#4A90E2) to convey professionalism and trust. Derived from the concept of fiscal responsibility, suggesting stability and security.
- Background color: Very light gray (#F5F5F5) for a clean and unobtrusive backdrop.
- Accent color: A slightly brighter, more saturated blue (#6CB4FF) to highlight key interactive elements. Chosen to complement the primary blue while drawing attention to important areas.
- Body and headline font: 'PT Sans', a modern sans-serif, for readability.
- Use minimalist line icons to represent categories and actions.
- Use a grid-based layout for a clean and organized presentation of data.
- Subtle transitions and animations for a smooth user experience.
